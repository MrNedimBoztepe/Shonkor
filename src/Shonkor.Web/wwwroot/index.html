How Plugins Work
  • A plugin is a pre-built assembly in a ZIP with a plugin.json
  • Each parser implements IFileParser
  • No runtime compilation — installing runs nothing
  • Plugins are inert until activated; activation is the trust gate
  • Active plugins load on the next workspace scan
